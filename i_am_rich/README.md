## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/l1nuxd/i_am_rich.git
   cd i_am_rich
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
